Steps for Integration

Each integration between Revinate and a Hotel PMS takes place in the same basic steps. Variations exist by PMS or integration type, but the base process is always the same.

Steps

Use this quick guide to create the relevant subtasks (as an Integrations Specialist) or to keep your customer advised of progress (as an Implementation Manager). Each of the below can be used as a reference for the process, and bolded portions can be used to create JIRA Subtask titles. The original ticket that these subtasks are created for should be in THIS format. See image

required fields

  • Project: Integrations (INT)
  • Issue Type: New Customer
  • Summary: [Specify the property and Revinate products associated with this new customer ("Hotel Emma - RM, RS"); RM=Revinate Marketing; RS= Revinate Surveys]
  • PMS: [Select the property management system used by the property]
  • Due Date: [Date when the Integrations work is due. This is typically two weeks before the launch date. Integrations Specialists use this field to prioritize their work.]
  • Linked Issues: related to
  • Issue: [Link the INT ticket to the ONB ticket]

Marketing Integrations

docs_1 an 'INT' JIRA ticket is filed for a SmartSync setup, the receiving Integrations Specialist tracks progress through the integration:

  • Establish PMS connection for [Name of Hotel]
  • Make connection with the Hotel PMS
  • Obtain test message(s) through the connection to verify
  • Follow the following steps for SmartSync integrations, provisioning, sending & setup of device:

    1. docs_1 an 'INT' JIRA ticket is filed for a SmartSync setup, the receiving Integrations Specialist creates a series of subtasks to track progress through the integration:
    2. Provision SmartSync for [Name of Hotel]:
    3. Assemble device
    4. Prepare and box for shipping
    5. Ship SmartSync device + directions to [Name of Hotel] and obtain SQL Server Credentials:
    6. Ship packaged device to hotel
    7. Send instructions
    8. Add SmartSync shipping information to the Pi Tracking: Doc
    9. Collect SQL Credentials to log in to PMS. Guide is here
  • Finish SmartSync setup after receipt by [Name of Hotel]
    1. Verify device checks in.
    2. Plug in SQL Credentials to connect to PMS.
    3. Here is a JIRA template to use to record the correct SQL Credential information docs_1 verified (paste into the JIRA subtask Description).

    5. Here's a JIRA Template to use for displaying data about finishing SmartSync setup/establishing the PMS connection:
      ||Object created||Link||Screenshot|| |Remote System|[|| |Mercer Property|[||
